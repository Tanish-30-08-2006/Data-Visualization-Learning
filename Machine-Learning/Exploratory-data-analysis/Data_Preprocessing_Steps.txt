
DataProcessing -> Transforming valid data into usable format so it can be further used for analysis or 
                  can be used in Machine Learning models

1. Encoding Categorical Variables      -> Convert Text labels ('male','yes', etc.) into numbers
                                       -> ML models generally fail to read text format data to some extent
                                          so we encode it into numbers

                                       -> Two Common Methods of Encoding Categorical Variables  :
                                             -> Label Encoding - Good for ordered categories like high/low
                                                                 [Ordinal Encoding] 
                                             -> On-Hot-Encoding - Good for Unordered Categories like 4 regions
                                                                  North/South/East/West
                                                                  [ Nominal Encoding ]

2. Feature Tranformation                -> Log/Square Root etc.
                                        -> Used to handle skewed data like left skewed or right skewed 

3. Feature Scaling                      -> Normalization / Standardization
                                        -> Bring Numerical Values to the Same Scale
                                        -> for eg : Bring all columns to same scale for scatter plot
                                        -> Especially Used for Distance-Based Algorithms

                                        -> Normalization (Min-Max Scaling) : Between 0-1
                                                                           : Used less in ML models
                                                                           : Used in pixels (Image Processing)
                                        -> Standardization(Z-Score Scaling) : Data scaled between -3 to +3
                                                                            : To have mean 0 / Std dev = 1

4. Feature Engineering                  -> Creating useful features(columns) or transform existing ones 
                                        -> For ML models to learn better from these features..
                                        -> Used as hit and trial to increase accuracy

                                        -> Common Feature Engineering Techniques In ML :
                                             -> Target-Based Flags  
                                             -> Binning (Eg: 10-20 age group of 10)
                                             -> Time based Features (If Time Exists in data)
                                             -> Mathematical Combinations 


5. Feature Selection                    -> Selecting most Useful Features and removing rest (Reducing Columns)
                                        -> Why : To Reduce Noise and Overfitting 
                                               : Improves Model Accuracy
                                               : Speeds up Training 

                                        -> Methods of Feature Selection : 
                                                 -> Correlation Matrix : Remove highly Correlated Features
                                                 -> Chi-Square Test (Categorical vs Categorical)
                                                 -> ANOVA F-Test (Numerical vs Categorical Target)
                                        -> Embedded Methods : 
                                                 -> Lasso Regression : Shrinks Coefficients to 0 
                                                 -> Tree Based Model : Random Forest , XGBoost etc.
